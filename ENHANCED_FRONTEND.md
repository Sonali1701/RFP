## 🔧 **INSTALLATION & SETUP**

### **1. Install Enhanced Dependencies**
```bash
pip install -r requirements-enhanced.txt
```

### **2. Start Backend**
```bash
python ultra_simple.py
```

### **3. Start Enhanced Frontend**
```bash
streamlit run enhanced_frontend.py
```
